# 📱 Quick Installation Guide - Visual Steps

## 🍎 iPhone/iPad (iOS) - 4 Simple Steps

### Step 1️⃣: Open in Safari
```
Open Safari → Go to:
https://car-maintenance-78.preview.emergentagent.com
```

### Step 2️⃣: Tap Share Button
```
Look at the bottom of Safari
Tap the [📤] Share icon
```

### Step 3️⃣: Add to Home Screen
```
Scroll down in the share menu
Find "Add to Home Screen" [➕□]
Tap it
```

### Step 4️⃣: Confirm & Launch
```
Tap "Add" (top right)
Go to Home Screen
Tap the app icon 🚗
Done! ✨
```

## 🤖 Android - 3 Simple Steps

### Step 1️⃣: Open in Chrome
```
Open Chrome → Go to:
https://car-maintenance-78.preview.emergentagent.com
```

### Step 2️⃣: Install
```
Look for "Install app" banner at bottom
OR
Tap [⋮] Menu → "Install app"
```

### Step 3️⃣: Launch
```
Tap "Install" to confirm
Find icon on Home Screen 🚗
Tap to open
Done! ✨
```
